> Macaca reporter used for jest.

## Installment

```bash
$ npm i macaca-reporter-jest --save-dev
```

## Configuration

defined in `jest.config.js`:

```bash
  ...
  reporters: [
    'default',
    ['macaca-reporter-jest', {}],
  ],
  ...
```

macaca-reporter-jest is based on [macaca-reporter](https://macacajs.github.io/macaca-reporter).
